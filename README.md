## Dataset
Global Urban Air Quality Index Dataset (2015–2025)
## Problem Statement

This project analyzes global air quality data to understand AQI trends, identify important pollutants, classify AQI categories using machine learning, and discover pollution patterns using clustering and PCA.

## Tools and Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## How to Run

1. Download the repository
2. Open Jupyter Notebook
3. Run AQI_Data_Science_Project.ipynb

## Main Findings
- PM2.5 showed strong relation with AQI
- KNN performed better than Naive Bayes
- K-Means identified pollution groups
- PCA helped visualize pollution patterns
